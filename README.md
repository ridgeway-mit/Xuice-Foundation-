# Xuice-Foundation-

To set up your local project directory and push it to a new GitHub repository, run:

> These commands assume the local directory is already a Git repository.
> Replace the local path and repository URL values with your own if they differ.

```bash
cd /path/to/your/mediapipe_project
git remote add origin https://github.com/YOUR_USERNAME/mediapipe-pose-tracker.git
git branch -M main
git push -u origin main
```
